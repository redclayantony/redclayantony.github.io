Determining Building Height Using LiDAR

Boca Chica, NASKW

One of the most important things to be aware of when working near an airfield is the height of the buildings and structures on or near the flight line. There are obstruction height regulations on any features that is within a certain distance to the airfield that if aren’t observed can shut down of operations pretty quick. I have used LiDAR data before to determine vegetation height so I knew it wouldn’t be hard to do the same for the buildings at my installation. We had some LiDAR data had been flown relatively recently but none of the processed files were loading properly so I had to start from scratch with the raw .las files.

First I did a little research and found this guidance from ESRI that I used as a reference to make sure I was doing everything correctly. My first step was to create a LAS dataset file (.lasd) to make it easier to access the individual .las files using ArcGIS's Create LAS Dataset geoprocessing tool. Once that was created I used the dataset file to create both a Digital Surface Model (DSM) using the first return points and a Digital Elevation Model (DEM) using the bare earth return points. Subtracting the DEM values from the DSM would give me a height of each pixel, but I was only interested in the height of my buildings, so I needed a way to extract the heights of just these features.

To do this I brought in my building and structure footprint vector file and used the Create Random Points tool to to create 20 random points within each polygon with a minimum spacing of 1-meter, the same as my LiDAR data resolution. From there I used the Add Surface Information tool and the DSM raster to determine the maximum elevation for each point. Next, I used the Summary Statistics tool can calculate the average of all the points within each building footprint in a seperate table.

I then repeated the Add Surface Information step, but this time using the DEM to create an additional table with the ground surface heights. Finally, I merged the two tables with the original footprint vector added a new field and calculated the difference between DSM and DEM elevations to get my building height.

As a bonus I joined the table back to the footprints layer and opened it in ArcScene and setup the extrusion based on height to create a 3-D visual representation of the buildings. And that was pretty much it, in less than an hour I had my building heights and could demonstrate that we were indeed in compliance.
