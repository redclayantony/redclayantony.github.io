Russell Lyon

GIS Analyst & Developer

I’ve been in this map game for over a decade. I created this site to highlight some of the projects that I have been a part of and show off some of my work. It was designed using several bootstrap templates that I customized to meet my needs.

Most of my GIS experience is with ESRI products, but for the last few years I have been using a lot more open-source applications like QGIS and GRASS. Almost all of my professional work is in ArcGIS, but at home I mainly rely on QGIS and its plugins. I have found that I can do just about everything that I need with it without the cost of ArcGIS.

I program mainly in Python and am most familiar with the ArcPy site package because I use its functions to automate my ArcMap workflows. Lately, I’ve been focusing more on JavaScript and web mapping using Leaflet and the Web Maps page has a few examples that I designed in my spare time. I am self-taught programmer learning mostly through web tutorials and sites like Codecademy. This site is hosted on Github and in setting them up I have learned Git and how to create and merge branches.

Thanks for taking the time to check out my work and get to know me a little better. If there is anything else you would like to know just shoot me an email.
