Georeferencing Scanned As-Builts & Digitizing Utilities

Mole Pier, Naval Air Station Key West, FL

One of the recent projects I worked on was updating of all underground utilities at the Mole Pier in Key West. The Mole Pier is one of the deepest ports in the city where large warships and the occasional cruise ship can dock. This project introduced me to a lot of utility datasets that I had never worked with before including things like electric transformers, back flow preventers and sewage lift stations.

There are a ton of underground utilities that lead out to the pier, but the Navy had not been the best at keeping track of their previous projects. I was able to track down some scanned As-Built drawings from some previous work done at the pier, but without the original CAD files, I was left no other option but to georeference the drawings and heads-up digitize the different features that were visible.

Since the drawings were projected in State Plane, I first had to convert all the existing data from UTM. Next, I had to set the control points. The drawings had some locations with coordinates on them, but using these as my control points did not create a good overlay so instead I matched up features such as manholes that were visible in both the CAD and aerial imagery. In the end it produced a good enough product for the local Planners to be able to identify which assets might be affected as new projects are planned in and around the pier. (Image sources: US Navy)
