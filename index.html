Qgis2threejs plugin

US Total Population per County

The Qgis2threejs plugin is a quick way to create a really cool 3D data visualization and is super easy to use. For this project I downloaded Total Population/County data using the US Census Bureau's American Fact Finder, then joined it to a US County shapefile and classified it based on population. Once that was ready, I opened the plugin and set the DEM to a flat plane and extruded the county polygons based on the total population field and ran the tool to create the 3D map.
